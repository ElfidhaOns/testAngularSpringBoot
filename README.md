
# testAngularSpringBoot

Projet Angular avec Spring Boot

## Backend (Spring Boot)

1. Ouvrir un terminal et se placer dans le dossier `backend` :
cd backend
2. Lancer l'application Spring Boot :
mvn spring-boot:run

## Frontend (Angular)

1. Se placer dans le dossier `frontend` :
cd frontend
2. Installer les dépendances :
npm install
3. Lancer le frontend :
ng serve --o

L'application frontend sera accessible à l'adresse : `http://localhost:4200`
